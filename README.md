# choliss

gulp.js を使用した静的Webページ作成用ボイラープレート

超速リリースを略してチョリスと命名しました

## 動作確認済み環境
```
"node": "10.8.0",
"npm": "6.3.0"
```
MacOS 10.13.6 で動作確認しました。

## 利用方法

1.Git リポジトリをクローン
```sh
git clone https://github.com/shimooon/choliss.git
```

2.プロジェクトディレクトリに移動
```sh
cd choliss
```

3.パッケージをインストール
```sh
yarn or npm i
```
